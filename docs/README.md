## 归档规则（两台设备的 Claude 都遵守）

1. **概念/决策推理** → 学习笔记（一个主题一个文件，编号递增，同步更新技能清单索引）
2. **做了什么/遇到什么问题** → 开发进度记录（追加条目，不改历史）
3. **影响范围/风险/口径的正式变更** → 对应的项目管理文档就地修订
4. **面试可用素材** → 作品集索引里登记状态（✅/🔶/❌）
