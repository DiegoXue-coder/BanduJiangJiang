隐私政策

伴读讲讲 · 最后更新：2026年6月
核心承诺：我们不收集、不存储、不出售你的任何个人数据。

1. 扩展收集哪些信息

伴读讲讲不收集任何可识别个人身份的信息。具体来说：

  • API Key：你填写的 DeepSeek、SiliconFlow、微信读书 API Key 仅存储在你本地浏览器的 chrome.storage.local 中，不会上传到任何服务器。
  • 划选文字：你在微信读书中划选的段落文字，仅在你主动点击「讲讲」时发送到后端处理，处理完成后不做任何存储。
  • 问答记录：你的提问和 AI 回答会保存在云端数据库，用于提供历史记录和语义记忆功能。这些数据不与任何个人身份关联。

2. 剪贴板权限说明

扩展申请了 clipboardRead 权限，用途如下：

当你在微信读书中划选文字并点击工具栏中的「讲讲」按钮时，扩展通过读取剪贴板获取你划选的内容，并将其发送给 AI 进行解释。

我们不会在其他任何时机读取你的剪贴板内容。

3. 数据如何传输

你的问题和书本上下文通过 HTTPS 加密传输到后端服务器（托管在 Railway），用于调用 AI 接口生成回答。传输过程全程加密。

4. 第三方服务

扩展依赖以下第三方服务，你使用这些服务时受其各自隐私政策约束：

  • DeepSeek — AI 对话
  • Microsoft Edge TTS — 语音合成
  • SiliconFlow（可选）— 语音识别

5. 数据安全

你的 API Key 只存储在本地，我们无法访问。问答记录存储在受保护的云端数据库中，不对外开放。

6. 儿童隐私

本扩展不面向 13 岁以下儿童，我们不会故意收集儿童的个人信息。

7. 隐私政策变更

如有重大变更，我们会更新本页面顶部的日期。建议定期查看。
